Astra Engine_

2025-06-20

Project Link: https://github.com/0x00ASTRA/astra-engine

Language: Zig

What Is It?

Astra Engine is a passion project of mine, focused on learning game engine design and implementation. It's a simple engine that supports multi-window rendering and a sweet lua scripting system. The project is still very much a work in progress and I have plans to expand in the future. Currently it only supports 2D but there is scaffolding written for 3D at a later time.

Features

  • Written in Zig
  • SDL Renderer
  • Multi-Window Support
  • Concurrency
  • Lua Scripting

Purpose

I wanted to understand the deep principles of game engine architecture and design. This project taught me so much about how games and engines function. I also longed to have a simple engine that minimizes depencies and bloat while offering Lua scripting for runtime logic.
