Getting Started With raylib and Zig_

2025-05-20

Introduction

This guide will help you get a basic project setup using the zig programming language with raylib-zig as a dependency. From there you will create a basic window using raylib and render some text. This is meant to be a very simple guide just to get a bare minimum project working.

Project Setup

To get started making our own game using Zig and raylib, we will need to ensure our environment is configured properly and project is initialized correctly.

Update Zig

First ensure that you're using an updated version of Zig or that your version will work with the raylib-zig dependency.

zvm i master && zvm use master

If you want to follow along exactly with me at the time of writing this, then you can use:

zvm i 0.14.0 && zvm use 0.14.0

Initialize the project

Once that's done create your project dir and initialize it. (Replace raylib_zig_game with whatever you want your project name to be).

mkdir raylib_zig_game
cd raylib_zig_game
mkdir src/
touch build.zig && touch src/main.zig

raylib-zig dependency

We also need to fetch the raylib-zig dependency:

zig fetch --save git+https://github.com/Not-Nik/raylib-zig#devel

Or if you're following along exactly with me:

zig fetch --save git+https://github.com/Not-Nik/raylib-zig?ref=devel#3bf08a304cfe2baf839705063ff999b8d8bc8c54

Configure build script

Now we need to configure our build script to actually build the project and link raylib-zig. Open build.zig in your editor of choice, (e.g., nvim build.zig), and set it up like this:

build.zig:

const std = @import("std");
// import raylib-zig build script. This is needed to set gl version
const rlz = @import("raylib_zig"); 

pub fn build(b: *std.Build) !void {
    const target = b.standardTargetOptions(.{});
    const optimize = b.standardOptimizeOption(.{});

	// Create the executable
    const exe = b.addExecutable(.{
        .root_source_file = b.path("src/main.zig"),
        .name = "game",
        .target = target,
        .optimize = optimize,
    });

	// Add raylib_zig as a dependency
    const raylib_dep = b.dependency("raylib_zig", .{
        .target = target,
        .optimize = optimize,
        .shared = true,
        // set open version for compatibility. This is for if GLFW fails to get an openGL context even though you have a newer GL version. Try building without this set.
        .opengl_version = rlz.OpenglVersion.gl_2_1,
    });

    const raylib = raylib_dep.module("raylib");
    const raygui = raylib_dep.module("raygui");
    const raylib_artifact = raylib_dep.artifact("raylib");

	// link raylib to our executable
    exe.linkLibrary(raylib_artifact);
    exe.root_module.addImport("raylib", raylib);
    exe.root_module.addImport("raygui", raygui);

    b.installArtifact(exe);
    
	// add a run step
    const run_exe = b.addRunArtifact(exe);
    const run_step = b.step("run", "Run the application");
    run_step.dependOn(&run_exe.step);
}

Setup the source files

Now that our build script is configured and raylib-zig is a module, we can get a basic window created to ensure everything is working. To do so, open src/main.zig in your editor of choice and follow along:

Basic Window

We will create the window using raylib and draw some text to the screen.

src/main.zig:

const std = @import("std");
const rl = @import("raylib");

pub fn main() anyerror!void {
	const screen_w: i32 = 800;
	const screen_h: i32 = 600;
	const target_fps: i32 = 60;

	// initialize the window
	rl.initWindow(screen_w, screen_h, "<Your Application Title Here>");
	defer rl.closeWindow();

	rl.setTargetFPS(target_fps);

	// main application loop
	while(!rl.windowShouldClose()) {
		rl.beginDrawing();
		defer rl.endDrawing();

		rl.clearBackground(.white);

		const cen_x: i32 = @divFloor(rl.getScreenWidth(), 2);
		const cen_y: i32 = @divFloor(rl.getScreenHeight(), 2);
		const font_size: i32 = 28;
		const font_width: i32 = @divFloor(font_size, 2);
		const font_color = rl.Color.black;
		const txt: [:0]const u8 = "I made a window with raylib-zig!";
		const txt_len: i32 = @intCast(txt.len);
		const txt_x: i32 = cen_x - @divFloor(txt_len * font_width, 2);
		const txt_y: i32 = cen_y - font_width;


		rl.drawText(txt, txt_x, txt_y, font_size, font_color);
	}
}

Build and Run

Now that we have our source file setup, we can build and run the project:

zig build run

This should open a window with 'I made a window with raylib-zig' drawn to the screen.

That's all for getting a basic window started. You can now extend this to create a full on game or application. Enjoy!
